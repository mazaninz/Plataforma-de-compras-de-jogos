# 🎮 GameHub — Plataforma Unificada de Compra de Jogos de PC
## 📖 Sobre o Projeto

GameHub é uma plataforma que centraliza a compra de jogos de PC de diferentes lojas digitais em um único ambiente.

O objetivo do projeto é eliminar a fragmentação do mercado de jogos, permitindo que jogadores encontrem, comparem e comprem títulos de diversos fornecedores em uma única interface, em vez de navegar entre várias lojas, o usuário pode utilizar o GameHub para visualizar jogos disponíveis em plataformas como:

* Steam

* Epic Games Store

* EA App

* Ubisoft Connect

* GOG

Assim, a plataforma funciona como um agregador de marketplaces de jogos digitais.

## 🚀 Principais Funcionalidades
Busca unificada entre jogos de diferentes plataformas simultaneamente

Comparação de preços 

Compras diretas pela plataforma

Sistema de avaliação da comunidade

Lista de jogos adquiridos

Wishlist: opção de salvar jogos para comprar futuramente

Mostrar diferentes promoções do mesmo jogo
